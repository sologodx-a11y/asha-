QUICK SETUP (FREE - NO ACCOUNT NEEDED):
======================================

1. Install Cloudflare Tunnel (cloudflared):
   - Download from: https://github.com/cloudflare/cloudflared/releases/latest
   - Extract and add to PATH, or use: choco install cloudflared

2. Start the Flask App FIRST:
   - Open a new command prompt
   - Navigate to: d:\asha
   - Run: python app.py
   - Keep this window open (Flask must be running)

3. Start the Quick Tunnel:
   - Double-click: start-cloudflare-tunnel.bat
   - The tunnel will automatically create a temporary public URL
   - Keep this window open (tunnel must be running)

4. Copy the URL:
   - Look for the URL in the command output (starts with https://)
   - Paste it above for future reference
   - Share this URL to access your site from any network

5. Access your site:
   - Use the URL provided in the command output
   - Example: https://random-name.trycloudflare.com

TROUBLESHOOTING:
================
If the site doesn't open when you share the URL:

1. Make sure BOTH windows are open:
   - Flask app window (python app.py)
   - Cloudflare tunnel window (start-cloudflare-tunnel.bat)

2. Check if Flask is running:
   - Open browser and go to: http://localhost:5000
   - If this doesn't work, Flask is not running properly

3. Check tunnel status:
   - Look for "Your quick Tunnel has been created" message
   - If you see errors, restart the tunnel

4. Try alternative - ngrok (easier):
   - Download from: https://ngrok.com/download
   - Extract and run: ngrok http 5000
   - Copy the https URL from ngrok window

5. Firewall issues:
   - Make sure Windows Firewall allows Python
   - Temporarily disable firewall to test
